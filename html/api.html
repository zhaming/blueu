Estimote灯塔是一个微型电脑

从技术上讲，Estimote灯塔是一个超级小型计算机。 它具有强大的32位ARM®的Cortex M0处理器有256KB闪存，加速度计，温度传感器，什么是最重要的 - 2.4 GHz的蓝牙4.0智能（又称BLE或蓝牙低功耗）的双向无线电。

但它是不一样的老蓝牙你的母亲会用（一个需要配对，从来没有真正的工作）。 这是一个完全新的，重新设计的标准由诺基亚™开发的，现在在所有的现代智能手机像苹果iPhone™4S或三星™的Galaxy S III，以及其他智能设备，如谷歌™玻璃，Fitbit追踪器或smartwatches实施。

这就是所谓的低能量，因为它确实是。 BLE设备可以运行长达2年以上与根据信号的抗剪强度，以及如何频繁，他们周围的广播信息的单个纽扣电池。

广播范围，并接近

你可以想想信标作为设备安装在一个固定的位置，并广播它的存在对周围所有的船只（智能手机）的小灯塔塔。 他们可能是短短的2英寸，至于230方呎（约为70米）。

确切的最大范围取决于过程的环境。 蓝牙是使用相同类型的无线电波为2.4GHz的无线路由器。 该信号可以被衍射，interferred或水（包括人体）吸收。

在该范围内即可拾取蓝牙无线电信号（无先前配对）和电话或其它智能设备估计的距离信标，通过测量接收信号强度（ RSSI ）。 你越接近灯塔，信号越强的。 取决于计划执行，设备可以探测信号的每个第二（1 Hz）或10次（10赫兹）。 更多的时候，它的完成，更稳定的信号和更敏感的应用是，并且可以实现更好的客户体验。

唯一标识符

智能手机可以从多个信标，同时拿起信号。 如果有范围内的三个或更多个信标，手机可以计算出的距离，每一个信标，并使用该数据来估计它的相对位置。 这种技术是什么，水手们用于年龄。 他们航行得益于独特的固定标志物像灯塔或星星。

Estimote信标可以被唯一地标识，以及因为每个信标广播其自己的ID。

地理围栏

现在是魔法的一部分。 应用程序可以在操作系统中（包括iOS和Android），不断监听特定的标识符订阅。 当手机进入了一个灯塔的范围内，应用程序被通知关于这一点，即使手机被锁定或应用程序当前没有运行。

监视距离和触发事件
